Acerca de mí

¡Hola! Soy Abraham Rodríguez, un Ingeniero en Sistemas Informáticos con pasión por el desarrollo de software. Me especializo en el desarrollo de aplicaciones web y móviles, con experiencia en diversas tecnologías como Java, JavaScript, Spring Boot y metodologías ágiles. Mi objetivo es crear soluciones eficientes y escalables que impacten positivamente en las empresas y usuarios.

Proyectos

Sistema de Inventario de Carnes

Proyecto en el que desarrollé una aplicación de inventario para la gestión de cortes de carne, control de reses y cálculo de ganancias, utilizando Spring Boot y Thymeleaf.

Gestión de Seguridad Informática

Un sistema para la administración de seguridad en aplicaciones web, con foco en protección de datos y autenticación, implementado en Java y utilizando mejores prácticas de seguridad informática.

Aplicación de Control de Costos

Herramienta que permite calcular los costos y ganancias de un negocio de venta de carne, utilizando C++ y bases de datos SQL para la gestión de inventarios y análisis financiero.
